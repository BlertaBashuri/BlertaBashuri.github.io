Blerta thinks about Bootstrap...

Bootstrap is a front-end library, which means it takes care of how our website looks in front of the user's eyes. Behind that beautiful website stand the codes that make possible how beautiful our website will look. Beautiful user interface like animation or buttons in the front end. Bootstrap is an open-source toolkit for developing with HTML, CSS, and JS. Through Bootstrap, we get access to pre-styled Elements that we can copy and paste into our code. Bootstrap is very firmly in the front end. It is free and open-source so that we can see all the source code. Codes are written to be reusable that we can drop into your site will make designing the website much easier for us. Make the website responsive to different viewports, which means if we are looking at the same website on a mobile or iPad or Desktop, it should have a different layout. It takes advantage of the size of the screen; I would call it ADAPTABLE Layout. A website changes its design depending on where it's being displayed.
